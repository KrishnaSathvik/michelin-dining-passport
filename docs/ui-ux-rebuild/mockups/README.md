## What to approve against

1. Pure white page background (not cream / warm near-white)
2. Named restaurants: photo only if approved, else designed fallback — no misleading stock
3. Explore: horizontal filters + All filters (no default left sidebar)
4. Map: 420px-class list **left**, map **right**, full viewport, no footer

## Review notes

- Generated frames may show placeholder product names or review-style star ratings — production uses Michelin star marks and this product’s brand.
- Named-restaurant photography in mockups is illustrative; implementation must follow [image-strategy.md](../image-strategy.md).

**Do not start `ui-ux-rebuild` until these six are approved and Phase 6 is on `main`.**
